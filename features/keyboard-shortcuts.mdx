## Global Shortcuts

These shortcuts work system-wide and are the fastest way to access AI Thing:

| Command                  | Description           |
|--------------------------|------------------------|
| `Control (⌃) + Space`    | Show or hide AI Thing  |

## Tab Management

Easily manage multiple tasks in parallel by creating and navigating between tabs:

| Command                  | Description           |
|--------------------------|------------------------|
| `Control (⌃) + N`        | Open a new tab         |
| `Control (⌃) + W`        | Close the current tab  |
| `Control (⌃) + >`        | Switch to the next tab |
| `Control (⌃) + <`        | Switch to the previous tab |

## Settings & Help

Quickly access key areas of the app:

| Command                  | Description               |
|--------------------------|----------------------------|
| `Control (⌃) + S`        | Show or hide Settings      |
| `Control (⌃) + H`        | Open Help panel            |

### Step Uno: Get

<AccordionGroup>
  <Accordion icon="download" title="Download AI Thing">
    <Card title="Download AI Thing" icon="rocket" href="https://get.aithing.dev" horizontal>
      Version 1.1 (2025-08-03)
    </Card>
  </Accordion>
  <Accordion icon="floppy-disk" title="Install AI Thing">
    1. Go to your **Downloads** folder and double-click the `AI Thing.dmg` file  
    2. Drag **AI Thing** into the **Applications** folder  
    3. Open the **Applications** folder and double-click the **AI Thing** app  
    4. Show/hide **AI Thing** using `Control (^) + Space`  

    <Warning>You may need to allow apps from unidentified developers. [Here's how](https://support.apple.com/en-us/102445)</Warning>
  </Accordion>
</AccordionGroup>

### Step Dos: Set

<AccordionGroup>
  <Accordion icon="brain" title="Choose Brain">
    1. Open **Settings** with `Control (^) + S`  
    2. Get your [Anthropic API key](https://docs.anthropic.com/en/api/overview#accessing-the-api)  
    3. Paste the key (starting with `sk-ant-...`) into the field shown below:  
       ![API Key Input](/images/quickstart-api-key.png)

    <Info>The free version doesn’t include one-click brain integration, so you'll need to enter the API key manually.</Info>  
    <Tip>Upgrade to the [Individual Plan](/plans#individual) to skip the API key setup.</Tip>
  </Accordion>

  <Accordion icon="github" title="Set Up GitHub Agent">
    The GitHub Agent lets you perform [these tasks](https://github.com/github/github-mcp-server?tab=readme-ov-file#available-toolsets).

    1. Go to the **Agents** tab in Settings  
    2. Click `+ Add Agent`  
    3. Enter the following details ([What are these?](https://github.com/github/github-mcp-server/blob/main/docs/installation-guides/install-other-copilot-ides.md#remote-server-recommended)):  
        - **Name:** `GitHub`  
        - **URL:** `https://api.githubcopilot.com/mcp/`  
        - **Auth Token:** `ghp_xyz...` ([How to create one?](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens#creating-a-personal-access-token-classic))  
    4. Close Settings with `Control (^) + S`

    You'll see a “Waking up agents...” popup. Once it disappears, you're good to go!

    <Tip>Skip manual setup with the [Individual Plan](/plans#individual).</Tip>
  </Accordion>
</AccordionGroup>

### Step Tres: Go

<AccordionGroup>
  <Accordion icon="rocket" title="What is @this">
    This isn't related to the GitHub Agent—just a quick example of how `@this` works.

    1. Open the AI Agent with `Control (^) + Space`  
    2. Type: `What is @this?`

    <Info>
      You'll see a small screenshot of your current screen. AI uses that screenshot to understand context.  
      Remove `@this` from your message to discard the screenshot.  
      Once received, the AI remembers the screenshot until the tab is closed.
    </Info>

    3. Hit **Enter**
  </Accordion>

  <Accordion icon="github" title="Summarize GitHub Issue">
    Let’s summarize an issue.

    1. Open the AI Agent with `Control (^) + Space`  
    2. Type: `Summarize https://github.com/WeAreAIThing/help.aithing.dev/issues/2`

    <Info>If it doesn’t work, make sure the "GitHub" agent is enabled in Settings.</Info>
  </Accordion>
</AccordionGroup>
